# Table 表格

使用场景：将数据呈现为高度可定制和具备可访问性的 HTML 表格，其核心功能为将结构化的数据使用表格的方式展现，然后可以使用各种参数来向表格中加入一些特性，比如排序，过滤，滚动，锁列等。

已知问题

分组 Table 不支持在 Hover 状态和选中状态下显示背景色，无法合并单元格

分组 Table 不支持在 Hover 状态和选中状态下显示背景色，无法合并单元格
